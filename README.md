
# Linux Fundamentals Labs

This repository contains my Linux practice exercises from TryHackMe.

## Skills covered
- File navigation
- Directory structure
- Basic Linux commands (ls, cat, cd)

Used `ls -l` to identify contents of directories and `cat` to read files.
 Commands Practiced
- `ls` / `ls -l` – list directory contents and file details
- `cd` – navigate between directories
- `pwd` – display current working directory
- `cat` – view file contents
- `mkdir` – create directories
- `touch` – create files
- `cp` – copy files and directories
- `mv` – move and rename files
- `rm` – remove files and directories
- `whoami` – identify the current user
- 
  Learning Outcomes
- Navigated the Linux file system using the command line.
- Identified files, directories, and permissions using `ls -l`.
- Read and analyzed file contents using `cat`.
- Created, moved, copied, and removed files and directories.
- Developed familiarity with common Linux commands used in cybersecurity and system administration
